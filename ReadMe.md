## 物理实验数据分析，可以求平均值，A类标准差之类的
实例：
```
======== 不确定度计算程序 (Word格式优化版) ========
输出格式特性：
1. 所有数学符号使用线性格式
2. 使用下划线表示下标 (如L_A)
3. 平方表示为^2，根号用sqrt()
4. 输入'quit'退出程序
=================================================

输入待测物理量符号 (如L、m): R

输入测量数据 [格式: n 数据1 数据2...]
或输入 quit 退出: 5 9.75        9.72    9.80    9.70    9.74

【平均值】
R_avg = (9.7500 + 9.7200 + 9.8000 + 9.7000 + 9.7400)/5 = 9.7420

【偏差计算】
ΔR_1 = 9.7500 - 9.7420 = 0.0080
ΔR_2 = 9.7200 - 9.7420 = -0.0220
ΔR_3 = 9.8000 - 9.7420 = 0.0580
ΔR_4 = 9.7000 - 9.7420 = -0.0420
ΔR_5 = 9.7400 - 9.7420 = -0.0020

【标准差】
σ =0.0377

【异常检测】3σ范围: [9.6290, 9.8550]
R_0 = 9.7500    [正常]
R_1 = 9.7200    [正常]
R_2 = 9.8000    [正常]
R_3 = 9.7000    [正常]
R_4 = 9.7400    [正常]

【结果输出】
R_A = σ = 0.0377
===============================

输入待测物理量符号 (如L、m): 
......
```
